隐私权政策

生效日期：2026年4月30日

数据收集声明

MarkDownload（Markdownit）不会收集、存储或传输任何用户个人数据。

我们访问的数据

本扩展仅在用户主动使用时，临时访问以下数据用于功能实现：

数据类型 用途 是否存储/传输
网页内容（文字、图片） 转换为 Markdown 格式 ❌ 仅本地处理，不上传
网页 URL 生成 Markdown 链接 ❌ 仅本地处理，不上传
用户设置（选项配置） 保存用户偏好 ✅ 仅存储在浏览器本地存储

我们不收集的数据

  • ❌ 身份验证信息（密码、PIN等）
  • ❌ 个人通讯（邮件、聊天记录）
  • ❌ 位置信息（IP、GPS等）
  • ❌ 网络记录（浏览历史）
  • ❌ 用户活动（点击、鼠标轨迹等）

数据处理方式

  1. 本地处理：所有网页内容转换均在用户浏览器本地完成
  2. 本地存储：用户设置仅保存在浏览器本地存储中
  3. 无第三方传输：不会将任何数据传输到外部服务器
  4. Obsidian 集成：仅在用户主动选择时，通过本地协议（obsidian://）打开 Obsidian 应用

权限说明

本扩展申请的权限仅用于功能实现：

  • activeTab：访问当前标签页内容以转换为 Markdown
  • downloads：下载生成的 Markdown 文件
  • storage：保存用户设置
  • clipboardWrite：复制 Markdown 到剪贴板
  • contextMenus：提供右键菜单功能
  • scripting：读取页面内容
  • offscreen：处理文件下载

开发者承诺

  • 不会向第三方出售或传输用户数据
  • 不会将用户数据用于与产品功能无关的目的
  • 不会使用用户数据进行信用评估或贷款相关活动
